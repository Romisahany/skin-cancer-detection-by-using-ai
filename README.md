Skin Cancer Detection Using AI

An AI-powered web platform for early skin cancer detection and classification using Deep Learning. The system enables users to upload skin lesion images and receive instant predictions, helping support early diagnosis and improve healthcare accessibility.


Project Overview

Skin cancer is one of the most common types of cancer worldwide. Early detection significantly increases the chances of successful treatment.

This project combines Artificial Intelligence with Web Development to provide an intelligent platform capable of classifying skin lesions from uploaded images.

The project was developed as a Graduation Project by students of the Network Technology Department.

 Features

- AI-based skin cancer classification
- Upload skin lesion images
- Instant prediction results
- Secure authentication system
- User-friendly dashboard
- Database integration
- Responsive web interface
- Successfully tested in a hospital environment

 AI Model

The classification model is based on **EfficientNetB3**, a deep learning architecture that provides high accuracy while maintaining computational efficiency.
